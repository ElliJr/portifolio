COFFE - Site

Um site focado em destacar os benefícios e a cultura do café, com um layout acolhedor e visual moderno. Ele oferece seções com informações sobre produtos ou serviços relacionados ao café, com uma estética envolvente que usa cores terrosas.

Tecnologias usadas:

  • HTML5: Para a estruturação das páginas.
  • CSS3: Para criar um design aconchegante e agradável, com animações suaves.
  • JavaScript: Para dinamizar a experiência do usuário, como animações e interações.

CBF Engenharia

Este site apresenta informações sobre a empresa CBF Engenharia, destacando seus serviços e valores. Com uma estrutura limpa e profissional, o design transmite credibilidade e foco no setor de engenharia. A navegação é simples e direta, permitindo que os visitantes encontrem informações relevantes sobre a empresa.

Tecnologias usadas:

  • HTML5: Estrutura do site.
  • CSS3: Estilização, com ênfase em layout responsivo.
  • JavaScript: Adicionado para melhorar a interatividade e funcionalidades dinâmicas.

Pokédex

Uma aplicação interativa que lista e detalha Pokémon de forma dinâmica. O usuário pode navegar por uma lista infinita de Pokémon e acessar detalhes como tipos, habilidades e estatísticas. O site traz uma experiência única para os fãs, utilizando dados da PokéAPI para atualização em tempo real.

Tecnologias usadas:

  • HTML5: Para estruturar os elementos da Pokédex.
  • CSS3: Para estilização com temas inspirados no universo Pokémon.
  • JavaScript: Para realizar requisições à PokéAPI e criar uma experiência de navegação dinâmica.

FoodyNow

Um site dedicado a experiências gastronômicas, com foco em explorar diferentes pratos e receitas. A interface é moderna, com elementos visuais atraentes que destacam opções de comida e informações sobre cada prato.

Tecnologias usadas:

  • HTML5: Para estruturar o conteúdo e organizar as seções.
  • CSS3: Para criar um design moderno e responsivo, com foco em uma experiência visual marcante.
  • JavaScript: Para tornar a interação do site fluida, com elementos dinâmicos e animações.
